## 🚀 Quick Start

### Step 1: Get Your Frontend URL

**From Railway Dashboard:**
1. Go to https://railway.app
2. Select **`@hos-marketplace/web`** service
3. Go to **Settings** → **Domains**
4. Copy the public URL (e.g., `https://hos-world-production.up.railway.app`)

**Or from Railway CLI:**
```bash
cd apps/web
railway status
```

### Step 2: Open URL in Browser
